## Default Permission

Default permissions for the screen capture plugin

#### This default permission set includes the following:

- `allow-take-screenshot`
- `allow-take-screenshot-with-ocr`
- `allow-start-screen-capture`
- `allow-stop-screen-capture`
- `allow-get-active-window-info`
- `allow-get-screen-capture-state`
- `allow-save-screenshot`
- `allow-search-screenshots`
- `allow-get-recent-screenshots`
- `allow-get-screenshot-image`
- `allow-get-screenshot-by-id`
- `allow-delete-old-screenshots`
- `allow-delete-screenshot-by-id`
- `allow-delete-all-screenshots`
- `allow-save-screenshot-embedding`
- `allow-search-screenshots-semantic`
- `allow-screenshots-missing-embeddings`
- `allow-save-companion-session`
- `allow-get-recent-companion-sessions`
- `allow-delete-companion-session`
