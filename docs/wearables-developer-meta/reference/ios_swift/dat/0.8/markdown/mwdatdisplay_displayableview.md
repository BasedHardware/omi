# DisplayableView Protocol
Extends
Sendable
A protocol for views that can be displayed on glasses.
[FlexBox](https://wearables.developer.meta.com/docs/reference/ios_swift/dat/0.8/mwdatdisplay_flexbox) conforms to this protocol. Pass a [FlexBox](https://wearables.developer.meta.com/docs/reference/ios_swift/dat/0.8/mwdatdisplay_flexbox) to [Display.send(_:)](https://wearables.developer.meta.com/docs/reference/ios_swift/dat/0.8/mwdatdisplay_display#send) to show content on glasses.
## Signature
```swift
protocol DisplayableView: Sendable
```
