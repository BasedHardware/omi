# ViewComponent Protocol
Extends
Sendable
A protocol for components that can be arranged inside a [FlexBox](https://wearables.developer.meta.com/docs/reference/ios_swift/dat/0.8/mwdatdisplay_flexbox) layout.
All view component types ([FlexBox](https://wearables.developer.meta.com/docs/reference/ios_swift/dat/0.8/mwdatdisplay_flexbox), [Text](https://wearables.developer.meta.com/docs/reference/ios_swift/dat/0.8/mwdatdisplay_text), [Button](https://wearables.developer.meta.com/docs/reference/ios_swift/dat/0.8/mwdatdisplay_button), [Image](https://wearables.developer.meta.com/docs/reference/ios_swift/dat/0.8/mwdatdisplay_image), [Icon](https://wearables.developer.meta.com/docs/reference/ios_swift/dat/0.8/mwdatdisplay_icon)) conform to this protocol.
## Signature
```swift
protocol ViewComponent: Sendable
```
