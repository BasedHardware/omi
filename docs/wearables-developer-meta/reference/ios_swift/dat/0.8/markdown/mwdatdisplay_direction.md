# Direction Enum
Extends
Sendable
Layout direction for [FlexBox](https://wearables.developer.meta.com/docs/reference/ios_swift/dat/0.8/mwdatdisplay_flexbox) and container components.
Determines the main axis along which children are arranged.
## Signature
```swift
enum Direction: Sendable
```
