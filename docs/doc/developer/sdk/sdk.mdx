## Omi SDKs

The Omi SDKs make it easy to build on top of [omi AI wearable ](https://github.com/BasedHardware/omi)in different languages

![Promo Banner 2 Pn](/docs/docs/images/promo_banner_2.png)

## Languages Supported:

- [Swift SDK](./swift)
- [Python SDK](./python)
- [React Native SDK](./ReactNative)

### How it works:

- connects to any omi device
- transcription runs either localy or using Deepgram
- no dependency on omi infrastructure
