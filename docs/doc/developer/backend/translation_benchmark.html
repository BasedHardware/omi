Key Findings

European / Cyrillic Languages

  • German: 3.3B achieves 94% of Google — near-parity for production use
  • Russian: Steady improvement from 75% → 86%, usable at 1.3B+
  • Ukrainian (medium resource): Largest per-step improvement, 69% → 81%

CJK Languages

  • Japanese: Plateaus around 65% chrF++ across all sizes
  • Chinese: Modest improvement from 56% → 63%
  • For CJK parity with Google, evaluate dedicated models (MADLAD-400-3B, fine-tuned variants)

Recommendation

1.3B is the sweet spot for production deployment:
  • Fits on a single L4 GPU (24GB VRAM) with ample headroom
  • Achieves 74–90% of Google quality on European/Cyrillic languages
  • Only ~30% slower than 600M, but significantly better quality
  • 3.3B is marginally better on most languages but 2x slower — diminishing returns

Methodology

Corpus

WMT24 test sets via sacrebleu — 997 sentence pairs per language pair, no authentication required. WMT (Workshop on Machine Translation) test sets are the standard benchmark corpus for MT research.

Metrics

MetricRoleDescription
chrF++PrimaryCharacter 6-grams + word bigrams. Robust for CJK, morphologically rich languages.
BLEUSecondaryCorpus-level via sacrebleu (13a tokenizer). Widely understood but struggles with CJK.
COMETOptionalNeural metric (wmt22-comet-da). Highest human-judgment correlation, requires GPU.

Reproducibility

  • Google API responses cached (SHA-256 keyed) — no re-billing across runs
  • --model-name flag labels per-model output files
  • Dry-run mode validates dependencies before incurring costs
  • Script: backend/scripts/benchmark_translation.py

Running

# Prerequisites
pip install sacrebleu httpx google-cloud-translate

# Benchmark a single model
python3 backend/scripts/benchmark_translation.py \
    --nllb-url http://localhost:10150 \
    --model-name nllb-200-distilled-1.3B \
    --languages de,zh,ja,ru,uk \
    --output-dir /tmp/benchmark-results \
    --skip-comet

# Compare all 3 model sizes
for model in "nllb-200-distilled-600M" "nllb-200-distilled-1.3B" "nllb-200-3.3B"; do
    python3 backend/scripts/benchmark_translation.py \
        --nllb-url http://localhost:10150 \
        --model-name "$model" \
        --output-dir /tmp/benchmark-results \
        --skip-comet
done
