## Open Source Support

For open-source support, we recommend:

<Steps>
  <Step title="Search Existing Issues">
    Check [GitHub Issues](https://github.com/BasedHardware/omi/issues) to see if your question has already been answered
  </Step>
  <Step title="Ask on Discord">
    Join our [Discord community](http://discord.omi.me) for real-time help from other users and contributors
  </Step>
  <Step title="Open a New Issue">
    If you've found a bug or have a feature request, [open a new issue](https://github.com/BasedHardware/omi/issues/new) on GitHub
  </Step>
</Steps>
