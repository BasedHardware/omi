## Quick Specs

| Component | Specification |
|-----------|---------------|
| Processor | nRF5340 dual-core Bluetooth LE SoC |
| Wi-Fi | nRF7002 Wi-Fi 6 chip |
| Audio | 2x TDK T5838 top-port PDM microphones |
| Storage | 8GB NAND Flash (CSNP4GCR01) |
| IMU | LSM6DS3TR-C 6-axis |
| Battery | 3.7V 150mAh LiPo |
| Charging | BQ25101 charger IC, magnetic pogo pins |
| Form Factor | 25.5mm diameter wearable pendant |

## How to Use These Files

<Steps>
  <Step title="Order PCBs">
    Upload the Gerber ZIP files from [`electrical/*/gerbers/`](https://github.com/BasedHardware/omi/tree/main/omi/hardware/consumer/electrical) to your PCB manufacturer (JLCPCB, PCBWay, etc.)
  </Step>
  <Step title="Source Components">
    Use the [BOM CSV](https://github.com/BasedHardware/omi/blob/main/omi/hardware/consumer/bom/omi-bom.csv) with manufacturer part numbers (MPN column) to order from Mouser, DigiKey, or LCSC
  </Step>
  <Step title="Manufacture Enclosure">
    Use STEP files from [`mechanical/parts/`](https://github.com/BasedHardware/omi/tree/main/omi/hardware/consumer/mechanical/parts) — each subfolder indicates the manufacturing process (CNC, injection molding, SLA, silicone)
  </Step>
  <Step title="Assemble">
    Reference the assembly photos and exploded views in [`assembly/`](https://github.com/BasedHardware/omi/tree/main/omi/hardware/consumer/assembly)
  </Step>
  <Step title="Flash Firmware">
    Follow the [firmware compilation guide](/doc/developer/firmware/Compile_firmware) to build and flash
  </Step>
</Steps>
