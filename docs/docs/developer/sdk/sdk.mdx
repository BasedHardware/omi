## Omi SDKs

The Omi SDKs make it easy to build on top of omi in different languages

## Languages Supported:

- [Swift SDK](./swift.mdx)

- [Python SDK](./python.mdx)

- [React Native SDK](./ReactNative.mdx)

### How it works: 
- connects to any omi device
- transcription runs either localy or using Deepgram
- no dependency on omi infrastructure
