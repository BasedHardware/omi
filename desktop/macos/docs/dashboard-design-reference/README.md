# Proof-first Dashboard design reference

`05-wow-lab-v2.html` is the finalized HTML/CSS/vanilla-JavaScript reference for
the proof-first macOS Dashboard. It records the approved information hierarchy,
three-tier hero cascade, Home / Connect data / Features navigation, and content
direction used by the native implementation.

These files are documentation only. They are not part of the Swift package or
an application build target. Open `05-wow-lab-v2.html` directly to inspect the
reference. Links back to the earlier lab and local mock index are retained from
the source artifact for provenance, but those comparison files are intentionally
not copied into this directory.

The native app remains governed by `OmiColors.swift` and `OmiType.swift`; mock
CSS values are visual reference values, not an additional token system.
