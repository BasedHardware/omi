# PR Summary

Summarize the changes in this branch and propose a PR title and description.

## Tasks

1. Analyze all changed files and summarize the changes
2. Propose a clear, descriptive PR title
3. Write a PR description including:
   - What changed and why
   - Testing performed
   - Breaking changes (if any)
4. Suggest a review checklist based on the changes
